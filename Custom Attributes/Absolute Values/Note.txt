The following scripts are for converting all negative number values in the inspector into positive number values.
Please keep in mind that the script and or asset may not be up to date with Unity!

AbsoluteValue.cs
- This script can be placed anywhere in the project.

AbsoluteValuePropertyDrawer.cs
- Must be placed in a folder named "Editor".

=============================================================================================

To use this attribute simply put [AbsoluteValue()] beside the variable.

[AbsoluteValue()] public int a;
